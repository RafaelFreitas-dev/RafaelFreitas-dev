Desenvolvedor Full-Stack em desenvolvimento focado na criação de interfaces dinâmicas, APIs robustas e arquiteturas eficientes de dados. Busco aplicar as melhores práticas de engenharia de software para desenvolver ecossistemas digitais escaláveis e completos, integrando aplicações modernas a backends ágeis e modulares.

- 💻 Construindo interfaces interativas e SPA modernas com **React**.

- 🗄️ Modelagem de dados ágil e segura com o **Prisma ORM**.
